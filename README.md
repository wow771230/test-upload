# 📝 待办事项 (Todo List)

一个简洁美观的待办事项网页应用，使用纯 HTML、CSS 和 JavaScript 构建。

## ✨ 功能

- ✅ 添加新的待办事项
- ✅ 标记完成 / 取消完成
- ✅ 删除单个待办事项
- ✅ 筛选（全部 / 进行中 / 已完成）
- ✅ 一键清除所有已完成事项
- ✅ 数据持久化（localStorage 本地存储）
- ✅ 响应式设计，支持手机和桌面端
- ✅ 优雅的渐变动画效果

## 🚀 使用方法

直接在浏览器中打开 `index.html` 即可使用。

## 🛠️ 技术栈

- HTML5
- CSS3 (Flexbox, 动画, 渐变色)
- JavaScript (ES6+)

## 📁 文件结构

```
├── index.html   # 主页面
├── style.css    # 样式文件
├── script.js    # 交互逻辑
└── README.md    # 项目说明
```
